# Choir Module — User Guide

## For all members

- **Welfare:** View open cases, contribute from case detail (web or mobile)
- **Music:** Browse library, favorite songs, view lyrics
- **Rehearsals:** See upcoming rehearsals and readiness (mobile/web)

## For coordinators / presidents

- **Welfare:** Create cases, review, record assistance, run reports
- **Rehearsals:** Open rehearsal from Events calendar → full workspace (plan, attendance, readiness)
- **Reports:** Choir reporting center — export PDF/CSV

## For treasurers

- Record welfare contributions (including anonymous)
- Export welfare case reports
- Contribution stewardship unchanged (Sprint 10)

## Languages

Switch locale in web URL (`/en`, `/fr`, `/rw`) or mobile Settings → Language.

## Offline (mobile)

After first online load, welfare list and song lyrics may display from cache when offline. Pull down to refresh when back online.
