# Glossaire terminologique (Français)

Le français doit sonner **naturel dans un contexte paroissial**, pas comme une traduction mot à mot de l'anglais.

## Rôles & personnes

| Anglais | Français (app) |
|---------|----------------|
| Choir | Chorale |
| Protocol | Protocole |
| Member | Membre |
| Leader | Responsable |
| Committee | Comité |
| Treasurer | Trésorier |
| Secretary | Secrétaire |

## Événements & planning

| Anglais | Français |
|---------|----------|
| Event | Événement |
| Worship service | Célébration / Culte |
| Rehearsal | Répétition |
| Schedule | Planning |
| Assignment | Affectation |
| Announcement | Annonce |

## Modules

| Anglais | Français |
|---------|----------|
| Attendance | Présence |
| Swap | Échange de service |
| Replacement | Remplacement |
| Discipline | Discipline |
| Choir finance | Finances de la chorale |

## Statuts de présence

| Statut | Libellé |
|--------|---------|
| Present | Présent |
| Absent | Absent |
| Late | En retard |
| Excused | Excusé |
| Unexcused | Non excusé |

## Workflow discipline

| Étape | Libellé |
|-------|---------|
| REPORTED | Signalé |
| UNDER_REVIEW | En cours d'examen |
| DECISION_PENDING | Décision en attente |
| ACTIONED | Mesure prise |
| CLOSED | Clôturé |

## Messages paramétrés (exemples)

- `swap_request_sent` : « {memberName} a demandé un échange de service avec vous »
- `attendance_marked_for_event` : « Votre présence pour {eventName} a été enregistrée »
- `discipline_case_opened` : « Un dossier disciplinaire a été ouvert : {caseTitle} »
- `dues_remaining` : « Solde restant : {amount} »
