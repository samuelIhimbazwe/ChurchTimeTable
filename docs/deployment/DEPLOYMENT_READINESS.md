# Deployment Readiness

## Levels

| Level | Typical score |
|-------|----------------|
| NOT_READY | &lt; 30% |
| PARTIAL | 30–54% |
| READY | 55–74% |
| PILOT_READY | 75–89% + setup completed |
| LIVE_READY | ≥ 90% + setup completed |

## Indicators

- Leadership configured  
- Members imported  
- Choirs configured  
- Protocol configured  
- Schedules created  
- Notifications active  
- Assets imported  
- Broadcasts configured  
- Ministries active  
- Imports completed

`GET /api/v1/setup/readiness`  
`GET /api/v1/setup/status`
