# Governance Alerts (MF-6)

Computed alerts — not persisted as separate alert rows in MF-6.

## Types

- `NO_LEADERSHIP`
- `NO_MEETINGS`
- `NO_REPORTS`
- `NO_ACTIVITY`
- `LOW_ENGAGEMENT`
- `INACTIVE_UNIT`
- `BUDGET_INACTIVE`
- `MISSING_SUCCESSION`

## API

`GET /church/intelligence/alerts`

Severity: `critical`, `warning`, `info`.
